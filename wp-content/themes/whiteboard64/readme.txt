Description: Whiteboard64 is a black and white, clean, minimalist and elegant Responsive News Magazine BLog Theme based on Bootstrap Framework. Theme is yet simple and easy to use for any type of blog magazine type online portal. It has plenty of white area to focus on the contents that enriched with the blog functionality and Ad Management Sections. Whiteboard64 theme has Top Header Ad section, five different Ad section for homepage tabbed news / blog section, archive / category Ad section and Single post Ad section. For demo: http://www.sumanshresthaa.com.np/theme/whiteboard64-theme.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== WHITEBOARD64 THEME HOMEPAGE ==
By default, WordPress site will takes you to Blog as your Homepage but in order to get the Whiteboard64 Theme Homepage, you have to go to Pages > Add New page. Here you can add title as Home and select the template as Homepage. Then Save the page, assign that page to frontpage via settings and all done...!!!


== WHITEBOARD64 THEME OPTION PANEL ==
	
Go to Appearance > Customize.
Click on Whiteboard64 Theme Option where you will get the following sections for the homepage to set.

*** Whiteboard64 Theme Option ***

1. Theme Logo:
You can replace the default theme logo with your own logo by uploading on Site Identity Section of Customizer. Uploaded image size must be 350px width and 60px height for the logo. After uploading your logo image, please unchecked at Display Site Title and Tagline to be replaced by logo image.


2. Social Media Section:
Here you can customize Social Section in Header Top Section. Make sure that you have filled all the social links, blank field will not be displayed in site and it will be hidden by default.


3. Homepage Slider Section:
Here you can customize Slider Section in Homepage. Make sure that slider images must be minimum of 1400px width and 550px height. The maximum no. of slides is 5.


4. Homepage Tabbed News / Blog Section:
Here you can customize News / Blog Section in Homepage. The maximum no. of tabs is 5. Choose the category for each tabbed news /blog from the select box that will be displayed in the Homepage accordingly.


5. Whiteboard64 Ads Management Section:
Here you can customize Ads for Header, Tabbed News / Blog Section, Archive / Category Page Section and Single Post Section. Here you can insert ads as images or iframe embeded as per requirements. If you are using image, the image size must be 730px width and 75px height for the Header Ad Section.
