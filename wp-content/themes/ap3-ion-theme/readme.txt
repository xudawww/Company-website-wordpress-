== Description ==
Description: The AP3 Ion theme is a theme specifically designed to be used in a native mobile app.

== Installation ==

1. Unzip the ap3-ion-theme.x.x.x.zip file. Be sure that the folder is named ion for updates.
2. Upload the ap3-ion-theme folder using your preferred FTP software to wp-content/themes/
3. Go to the AppPresser menu.
4. There is no need to select the theme as the AppPresser core plugin will select it for you when viewed in the app.
